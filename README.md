# 🌱 EcoTrack – Smart Sustainability Challenge

## Overview

EcoTrack is a sustainability-focused web application that helps users build eco-friendly habits through gamification, rewards, community engagement, and environmental impact tracking.

The platform encourages users to adopt sustainable practices such as water conservation, energy saving, recycling, and public transport usage while earning points, maintaining streaks, unlocking achievements, and competing with others through leaderboards.

EcoTrack transforms sustainable living into an engaging, measurable, and rewarding experience.
